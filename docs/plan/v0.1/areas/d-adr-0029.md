## Scope

Implement ADR-0029's two directional buckets:

1. **Agent SDK contract (platform → tenant):** Three lifecycle hooks — `initialize(context)`, `on_message(message)`, `on_shutdown(reason)` — exposed as an embeddable SDK. A2A 0.3.x is the wire.
2. **Public platform API (tenant → platform):** A2A send endpoint. Tenant containers call the platform's routing endpoint. Language-agnostic, semver-versioned.

`IAgentContext` bootstrap bundle, per-agent persistent filesystem volume, and platform actor supervision round out Stage 3. Stage 4 (memory contract) is **intentionally deferred** — blocked on Area F.
